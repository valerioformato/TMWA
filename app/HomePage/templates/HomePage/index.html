Welcome to TMWA

This is a web-based tool for monitoring the status of the AMS-02 silicon tracker in many different ways. It features:

  • A Long-term monitoring app, where data from the tracker calibrations can be plotted as a function of time, on a per-TDR basis, for both X and Y sides.
  • A Real-time monitoring app, which is basically a web-port of tkonline where the last calibration can be displayed and controlled.

Use the navigation bar at the top to select between one of the two monitors.

Please report any bug you encounter to the addresses in the 'Contacts' section.
